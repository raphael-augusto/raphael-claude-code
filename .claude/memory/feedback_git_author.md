Nunca usar Co-Authored-By do Claude em commits. Autor deve ser sempre Raphael.

**Why:** Usuario quer que todo historico de git e PRs (GitHub, Azure DevOps) fique no nome dele.
**How to apply:** Nao adicionar "Co-Authored-By: Claude" em commits. Usar apenas o git user configurado no sistema.
